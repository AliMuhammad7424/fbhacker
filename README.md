      $sudo apt install python3-pip
      $git clone https://github.com/H3inAK/fbhacker
      $cd fbhacker
      $pip3 install -r requirements.txt
      $python3 fbbrute.py
      
you can also set your own passlist by typing nano <filename.txt> and edit fbbrute.py by calling nano.
